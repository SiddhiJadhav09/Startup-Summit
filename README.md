# Startup Summit Expo 🚀

## Overview

Startup Summit Expo is a modern and responsive event website designed to showcase a startup exhibition. The project focuses on presenting event details, promoting innovation, and creating an engaging experience for visitors.

## Features

* Responsive and modern user interface
* Attractive landing page design
* Startup and innovation-focused sections
* Smooth navigation
* User-friendly layout

## Technologies Used

* HTML
* CSS
* JavaScript

## Live Demo

https://siddhijadhav09.github.io/Startup-Summit/

## Purpose

This project was developed to improve frontend development skills, UI/UX design understanding, and create an engaging website experience for an event-based platform.
